🎯 المبادئ الأساسية 7
================================================================================

1️⃣ الفحص الشامل (Row-by-Row Integrity)
   • كل صف يُفحص بشكل مستقل
   • بدون تجاهل أي قيمة
   • توثيق شامل لكل خطوة

2️⃣ التحقق من نوع البيانات (Deep Type Checking)
   في أعمدة التاريخ:
     - نص عادي → نقل لـ Notes
     - تاريخ صحيح → تنظيف وتحويل
     - null → الحفاظ عليه
   
   في أعمدة الأرقام:
     - نص → نقل لـ Notes
     - رقم صحيح → تحويل لـ Int64
     - رقم عشري → الحفاظ عليه

3️⃣ تطهير التواريخ (Date Sterilization)
   • إزالة أصفار الوقت (00:00:00)
   • تحويل لـ YYYY-MM-DD
   • توثيق أي وقت تم حذفه

4️⃣ منع الترحيل العشوائي (Data Shifting Prevention)
   • مقارنة البيانات الأصلية بالمعالجة
   • توثيق التغييرات غير المتوقعة
   • سجل تفصيلي لكل ترحيل

5️⃣ تنسيق المخرجات (Excel Formatting)
   • التواريخ: YYYY-MM-DD
   • الأرقام الصحيحة: بدون .0
   • الأرقام العشرية: 0.00

6️⃣ الأداء العالي (High Performance)
   • معالجة ملايين الصفوف بسرعة
   • استخدام vectorized operations
   • استخدام ذكي للموارد

7️⃣ التعليقات التفصيلية (Comprehensive Documentation)
   • كل دالة موثقة
   • شرح لكل خطوة
   • تعليقات عربية وإنجليزية

import pandas as pd
from comprehensive_processing_engine import ComprehensiveProcessingEngine

data = {
    'ID': ['P001', 'P002', 'P003', 'P004'],
    'Price': [999.99, '50.0', 'غير متوفر', 150.0],
    'Order_Date': ['2024-01-15', '2024-02-20 14:30', 'بدون تاريخ', '2024-04-10'],
    'Notes': ['', 'غير واضح', '', '']
}

df = pd.DataFrame(data)

engine = ComprehensiveProcessingEngine(
    df=df,
    date_columns=['Order_Date'],
    numeric_columns=['ID', 'Price']
)

result = engine.run_complete_pipeline(output_file='products.xlsx')
