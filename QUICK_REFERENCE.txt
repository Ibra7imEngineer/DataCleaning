## 🎯 Arabic Normalization

**What happens:**
- أ, إ, آ → ا
- ة → ه
- ى → ي
- Diacritics removed
- Whitespace stripped

**Example:**
```
ترابيزة → تربيزه
إسلام → اسلام
مُحَمَّد → محمد
```

---

## 🔍 How It Works

1. **Identify** valid text columns
2. **Skip** high-cardinality columns (IDs)
3. **For each column:**
   - Process from most → least frequent
   - Normalize text
   - Find similar values (fuzzy matching)
   - Unify under master word
   - Log corrections
4. **Return** cleaned DataFrame + logs
