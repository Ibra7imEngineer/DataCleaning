# AI Deep Clean - ذكاء اصطناعي لتنظيف البيانات

تطبيق ويب تفاعلي لتنظيف ومعالجة البيانات باستخدام الذكاء الاصطناعي

## المميزات

- تنظيف شامل للبيانات
- دعم اللغة العربية
- واجهة ويب سهلة الاستخدام
- معالجة متقدمة للتواريخ والأرقام
- تقارير مفصلة للعمليات

## التثبيت المحلي

```bash
pip install -r requirements.txt
streamlit run app.py
```

## النشر على Streamlit Cloud

### الخطوة 1: إعداد GitHub

1. أنشئ حساب على GitHub إذا لم يكن لديك
2. أنشئ مستودع جديد (Repository)
3. ارفع جميع ملفات المشروع إلى المستودع

### الخطوة 2: النشر على Streamlit Cloud

1. اذهب إلى https://share.streamlit.io
2. اربط حساب GitHub الخاص بك
3. اختر المستودع الذي أنشأته
4. اختر الفرع الرئيسي (main)
5. الملف الرئيسي: `app.py`
6. اضغط "Deploy"

### الخطوة 3: إعداد المفاتيح السرية

في إعدادات التطبيق على Streamlit Cloud:

1. اذهب إلى Settings > Secrets
2. أضف المفتاح التالي:

```
GROQ_API_KEY = "مفتاح_Groq_API_الخاص_بك"
```

### الخطوة 4: الحصول على مفتاح Groq API

1. اذهب إلى https://console.groq.com/
2. أنشئ حساب ومفتاح API
3. انسخ المفتاح وأضفه في الإعدادات

## المتطلبات

- Python 3.8+
- Streamlit
- Pandas
- NumPy
- وغيرها من المكتبات المذكورة في requirements.txt

## ملاحظات الأمان

- لا تقم بمشاركة مفاتيح API الخاصة بك
- الملف `.gitignore` يمنع رفع الملفات السرية
- استخدم دائماً Streamlit secrets للمفاتيح السرية
